Descripción del TFG

El proyecto consiste en una web de restaurante con un estilo visual similar a Foster Hollywood o Ginos.

Header (común a todas las páginas)

Logo y nombre del restaurante alineados a la izquierda.

Menú de navegación con las opciones:

Carta → redirige a una página con la carta completa.

Promociones → redirige a una página con promociones activas.

Contacto → despliega un formulario de contacto para enviar un email y, debajo, muestra el número de teléfono.

Acceder → al pasar el ratón por encima, se despliega justo debajo un menú con:

“Iniciar sesión”

“Registrarse”
Cada opción abre una ventana distinta (una para login y otra para registro).

Contenido principal (body)

Carrusel o sección dinámica con imágenes de platos y el restaurante de fondo.

Las imágenes estarán generadas con Gemini Pro y se alternarán automáticamente.

Cada imagen tendrá una frase descriptiva.

Al pulsar sobre un plato se mostrará una vista o modal con la explicación detallada del plato.

Botón visible para realizar una reserva, que redirige a una página donde se podrá:

Seleccionar fecha y hora.

Indicar número de personas.

Elegir la zona del restaurante.

Chatbot

Se integrará un chatbot para mejorar la experiencia de usuario.

Funcionalidades esperadas:

Responder preguntas sobre platos (ingredientes, alérgenos, guarniciones, etc.).

Permitir realizar reservas a través del chatbot.

Detectar si el usuario ya ha hecho reservas anteriormente y sugerir repetir una reserva (por ejemplo: “Ya hiciste una reserva anteriormente, ¿quieres repetir?”).

El chatbot se implementará utilizando n8n.

Se aceptan sugerencias técnicas o funcionales para mejorar el chatbot.
